Has method

Relates the contribution to types of methods that were used in the contribution. A contribution generally combines different methods to come up with a solution to a problem. This property shall be used to define the immediate key building blocks that were used within the contribution. (e.g. a contribution that aims to make sentiment analysis might use methods (immediate building blocks) such as: a transformer model as the training method, bag-of-words as the vectorization method or k-fold cross validation as the evaluation method etc.)
